# 記事掲載用ファイル

| ファイル | 用途 |
| --- | --- |
| [ARTICLE-zenn.md](ARTICLE-zenn.md) | Zenn 投稿用（front matter 付き、画像は GitHub raw URL） |
| [../ARTICLE-DRAFT.md](../ARTICLE-DRAFT.md) | リポジトリ内完結版（相対パス画像・Mermaid 図） |

## Zenn への投稿手順

1. Zenn で新規記事を作成
2. `ARTICLE-zenn.md` の本文をコピー（または Zenn CLI で `zenn new:article`）
3. `published: true` に変更して公開
4. 記事末尾にリポジトリ URL を掲載

## Qiita / 他媒体

`ARTICLE-DRAFT.md` から Mermaid 部分を画像に差し替えるか、テキスト説明に置き換えてください。
