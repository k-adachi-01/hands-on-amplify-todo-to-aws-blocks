## 🟠 重要: 発展編（第3章）の手順が薄すぎて再現できない

現状の発展編は実質これだけです。


初心者には次が不足しています（`from-chapter-2.patch` の中身を確認した上での指摘）。

1. **パッチ適用の実コマンドが無い**。「適用」とだけ書かれ、`git apply docs/chapters/04-chapter3-advanced/diffs/from-chapter-2.patch` のような具体例がない。パスもコマンドも初心者は分からない。

3. **依存追加 `@aws-blocks/bb-realtime` が必要**。`from-chapter-2.patch` は `aws-blocks/package.json` に `@aws-blocks/bb-realtime` を追加しており、`npm install` の再実行が要る。記事に無い。
